## 🚆 XPT 야간열차 정보

- **경로:** Sydney Central Station (20:42) → Melbourne Southern Cross Station (다음날 07:30)
- **클래스:** 슬리퍼 클래스 — 개인 침대칸 + 샤워·화장실 포함
- **장점:** 호텔 1박 절약 + 이동 시간 절약 + 뷔페차 운영
- **요금:** 1인 약 AUD $110~180
- **예약:** [NSW TrainLink](https://www.nswtrainlink.info/) — 사전 예약 필수
